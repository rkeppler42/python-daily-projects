## English

### About

Independent Python projects built from descriptions published on [Daily Python Projects](https://dailypythonprojects.substack.com/) by Ardit Sulce.

The approach: read only the project description, build the solution from scratch, then optionally compare with the author's solution. No peeking at the code beforehand.

### Structure

```
.
├── 01-project-name/
│   ├── main.py
│   └── README.md
├── 02-project-name/
│   └── ...
└── README.md
```

Each project folder contains its own `README.md` with a brief description and instructions to run it.

## Português

### Sobre

Projetos Python independentes construídos a partir das descrições publicadas na newsletter [Daily Python Projects](https://dailypythonprojects.substack.com/) de Ardit Sulce.

A abordagem: ler apenas a descrição do projeto, construir a solução do zero e, opcionalmente, comparar com a solução do autor ao final. Sem olhar o código antes.

### Estrutura

```
.
├── 01-nome-do-projeto/
│   ├── main.py
│   └── README.md
├── 02-nome-do-projeto/
│   └── ...
└── README.md
```

Cada pasta de projeto contém seu próprio `README.md` com uma breve descrição e instruções para executá-lo.
